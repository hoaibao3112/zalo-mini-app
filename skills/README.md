# Skills

Thư mục `skills` chứa các tài liệu tham chiếu dạng Markdown để lưu trữ kiến thức, hướng dẫn, và ngữ cảnh liên quan tới dự án.

Cấu trúc:

- `context/` — tài liệu mô tả ngữ cảnh, cấu trúc dữ liệu và các hợp đồng API nội bộ.

Sử dụng: lưu các file `.md` nhỏ, dễ đọc, có thể được AI hoặc người phát triển khác tham chiếu khi cần hiểu cấu trúc hệ thống.
