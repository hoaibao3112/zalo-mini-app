# Zalo Mini App

Dự án Zalo Mini App độc lập được tách ra từ hệ thống Sale Funnel.

## Cấu trúc thư mục
- `/frontend`: Mã nguồn ứng dụng Zalo Mini App (React + Vite + TailwindCSS + Konsta UI).
- `/backend-module`: Mã nguồn module APIs, Services, Middlewares và Validators của Mini App.

## Hướng dẫn chạy dự án
Xem hướng dẫn cụ thể bên trong từng thư mục.
